# Employee-Promotion-Prediction-System
Machine Learning project to predict employee promotion eligibility using Support Vector Machine (SVM).
## Overview
This project predicts employee promotion eligibility using Machine Learning and Support Vector Machine (SVM).

## Features
- Employee performance analysis
- Promotion eligibility prediction
- Data preprocessing and feature engineering
- Employee reports generation

## Technologies Used
- Python
- Flask
- Machine Learning
- Support Vector Machine (SVM)
- MySQL
- HTML
- CSS

## Project Outcome
The system helps organizations make data-driven promotion decisions and reduce manual bias in promotion processes.
